Pushnator

Pushnator: um projeto de automação de tarefas com python, criado durante o período de estágio na PGFN, o objetivo do software é entrar no site do PJE com username e senha, ler dados de uma planilha, incluir ou excluir processos, e por fim devolver outra planilha informando se os processos foram incluídos ou excluídos.

Repository

Zapbot

Zapbot: um projeto de automação de tarefas com python, criado durante o período de estágio na PGFN, o objetivo do software é entrar automaticamente no whatsapp web, maniplando o browser com a biblioteca selenium do python, e enviar mensagens, fotos e arquivos automaticamente para mais de um contato.

Repository

Gallery

Gallery: é uma página web, com um carrousel feito com HTML, CSS e JavaScript para baixar mais de 40 Wallpapers para desktop, foi criado durante o período da faculdade de Análise e Desenvolvimento de Sistemas.
